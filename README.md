# 이력서 메이커

내 모든 경력을 모은 마스터 이력서를 만들고, 회사별 맞춤 이력서를 뽑아주는 도구.

기능

- 마스터 이력서 만들기/갱신 — 내 경력 원본을 누적해서 모음
- 회사 맞춤 이력서 뽑기 — 채용공고(JD)에 맞춰 관련 내용만 골라 이력서 생성
- PDF로 빼기 — 이력서를 PDF로 변환
- 회사 검토 — 매출·영업이익·인원 보고 지원할지 판단

채팅에 그냥 말하면 알아서 동작함. (예: "마스터 이력서 만들어줘", "이 JD로 이력서 뽑아줘", "PDF로 빼줘", "이 회사들 걸러줘")

> 회사 검토에서 "원티드 북마크 검토" 기능은 **Claude for Chrome(브라우저 확장)이 연결돼 있고 원티드에 로그인된 상태**여야 동작함. 안 되어 있으면 회사명을 직접 붙여넣어 쓰면 됨.

## AI 동작 지침

이력서 관련 요청이 오면 `resume-orchestrator` 스킬로 시작해 조율한다. 스킬은 `.claude/skills/`에 있고, 상세 규칙·변경 이력은 `CLAUDE.md`를 따른다.

기능별 스킬:

- `ingest-resume` — 이력서 파일/텍스트를 경력·프로젝트·스킬·학력으로 파싱 (마스터의 입력단)
- `build-master-resume` — 파싱 결과를 `data/master-resume.md`에 누적 병합 (중복은 합치고 새 내용 추가)
- `tailor-resume` — JD에서 요구사항을 뽑아 마스터에서 관련 내용만 선별·강조해 맞춤 이력서 생성
- `export-pdf` — 마크다운 이력서를 PDF로 변환 (사용자가 명시 요청할 때만)
- `screen-companies` — 회사 매출·영업이익·인원을 조회해 지원/거름 판정 (직접 입력 또는 원티드 북마크 자동 수집)

핵심 원칙:

- 마스터에 없는 내용은 지어내지 않는다. 근거 없는 경력·수치는 금지.
- 개인 데이터는 `data/`에 두고 깃에 올리지 않는다 (`.gitignore` 처리됨).
- 결과물은 한 단계씩 보고하고, 다음 단계로 멋대로 넘어가지 않는다.
